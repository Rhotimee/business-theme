# business-theme


This is a free responsive theme designed for a business or a company. It is completely responsive and flexible for customisation.

This template is free, 100% responsive.

This template serves as a great marketing tool and can be used to keep your customers updated with your business and products. It has clear call-to-action to explain your business and what you do to your customers and prospective customers.


#### Usage
To use this template, You can clone it on your computer
```
$ git clone https://github.com/Rhotimee/my-portfolio.git
```
Edit it to your taste
Then host it.
